Intro to HTML

1. What is HTML?

HTML (HyperText Markup Language) is the backbone of every website. It provides structure and meaning to content on the web.

2. Basic HTML Structure

An HTML document starts with <!DOCTYPE html>, then includes <html>, <head>, and <body> sections.

3. Common Tags

<h1> for headings, <p> for paragraphs, <a> for links, <img> for images.

4. Creating Your First Page

Try building a simple “Hello, World!” webpage. Save it as index.html and open in your browser.
